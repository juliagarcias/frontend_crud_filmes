# Atividade Prática 2 - N2 - CRUD - FRONTEND - FILMES

**Realizada por: Julia Garcia e Jennifer Lopes**

Projeto Nextjs

Baixar o backend do projeto
- Configurar a porta para 3001

Para o front instalar as dependências
- npm i

Criar o arquivo .env na raiz do projeto:
- Adicionar a variável NEXT_PUBLIC_API_URL=http://localhost:3001

Para rodar o projeto:
- npm run dev
